trims



An open-source design production pipeline connecting designers and developers.

why make this?

A regular design collaboration at school always becomes a chaos of rambling files. Using email or web disks helps little except messing up the versions. And things could be worse when there are developers enrolled, especially for those using Windows because it is not possible for them to even open some design files such as .sketch.

As our first try of designer-oriented file management and version control, an open-source git solution GitLab was deployed on a test server in the college. It was significantly better when designers worked simultaneously in one well-managed git repository and developers were able to pull the latest assets so much easier. But the shortage of the code-oriented system is too obvious to ignore, such as designer-unfriendly interface and the low efficiency of the managing mechanism of binary files. The problem is still unsolved.

how will trims solve this?

Trims is designed to be a simple and powerful tool for designers and developers from ground up. An easy-to-use, cross-platform client enables designers to collaborate, roll back files, and hand over assets is bundled in the system with a powerful resource exporter built in.

A well-designed cross-platform client managing files and versions automatically with a built-in resource generator.

The intuitive design of Trims enables all the collaborators manage files and track versions in a much simpler way. And for anyone in the repository, we designed a powerful and easy-to-use synchronization mechanism handles binary file delta updating and conflict solving.


Inspect specs and export resources.

The resource inspector and generator built in Trims is designed for whoever cropping image resources and measuring the specs, especially developers. Developers can find and create just what they need easily without installing any complicated or platform-monopoly design software.

why open-source?

Just like GitLab, we want Trims not only to be a free toolkit but a base of collaboration for students and starters from different disciplines. Everyone can download and deploy Trims or even folk the project and modify it to make all good of it. And from 2015, there are already commercial products like Avocode for business or high-end users.

Follow the progress

Trims is still on the way to distribution. We moved the project to GitHub from the school SVN server so now everyone can clone or download the whole repository.
