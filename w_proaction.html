Proactive HMI System


An innovative car HMI system contains multi-sensor interior hardware and advanced infotainment software built for customer progressive feature test. Aimed to practice and evaluate our latest ideas of automobiles in the near future.

The project mainly includes the following phases:

Prophase Research

Customer survey, competitor review and cabin user interview.

HMI Design

Prophase Report analyse, information architecture, hardware and software interface design.

Test Platform Build

Specs documentations, software and hardware development, sandbox development.

User Simulator Test

Scenario-based user driving simulation, data collection and user interview, recursive report.
